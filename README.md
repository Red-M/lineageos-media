# LineageOS Media
Systemless replace current ROM's ringtones, notifications, system sounds and boot animation from LineageOS for another custom ROMs other than LineageOS. Feels Lineage-ify.

## Compatibility
- Android Nougat+ (API 24)
- SELinux Enforcing
